# Portfolio Asset Standard

Bu klasör proje bazlı görsel varlıklarını düzenli tutmak için kullanılır.

## Klasör Yapısı
- `project-01/`
- `project-02/`
- `project-03/`

Her proje klasörü en az bir hero görseli içermelidir.

## Dosya Adlandırma Standardı
- Hero web mockup: `hero-16x10@2x.<ext>`
- Detay ekranı: `detail-16x10@2x.<ext>`
- Mobil ekran: `mobile-9x16@2x.<ext>`

## Üretim / Export Standardı
- Ana oran: **16:10** (web mockup)
- Retina export: **2x** (ör. 1600x1000)
- Format önceliği: `webp` > `png` > `jpg`
- Renk profili: `sRGB`

## Önerilen Dosya Boyutu
- Hero görseli: 200-450 KB arası (web optimizasyonu sonrası)
